About This Tool

This tool uses FFmpeg.wasm to join video clips inside your browser. Stream Copy mode is fast and lossless — ideal when all clips share the same codec. Re-encode mode is universal — it re-encodes everything to H.264 MP4, handling mixed formats at the cost of processing time.

Frequently Asked Questions

Are my videos uploaded to a server?
No. All merging is done using FFmpeg.wasm which runs entirely in your browser. Your video files never leave your device and are never transmitted to any server.
Do videos need to be the same format?
For stream copy mode (fastest, no quality loss), videos should share the same codec and format. If they differ, the tool can re-encode them to a common format, which allows merging any combination of input formats.
What is the output format?
The merged output is saved as an MP4 file, which offers the best compatibility across devices, media players, and platforms.
