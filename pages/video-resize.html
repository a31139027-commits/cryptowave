About This Tool

This tool uses FFmpeg.wasm to resize video inside your browser using the scale filter. Select a common resolution preset (240p–4K) or type a custom width and height. Enable the aspect ratio lock to keep proportions correct when changing one dimension. Output is exported as MP4 (H.264).
