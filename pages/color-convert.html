About Color Formats

HEX — used in CSS and web design. RGB — mixes red, green and blue light (used in screens). HSL — intuitive model with hue, saturation and lightness (great for adjustments in CSS). HSV — similar to HSL but uses value instead of lightness (common in design apps). CMYK — the model used for color printing with ink.
