About This Tool

This tool uses FFmpeg.wasm to repeat a video N times inside your browser. It uses the concat demuxer with stream copy — no re-encoding — so the process is fast and the output is lossless. The result is a single MP4 file that plays the video back-to-back the chosen number of times.
