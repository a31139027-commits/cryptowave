About This Tool

This tool uses the Web Audio API to decode each audio file into raw PCM data, concatenate them in order (with optional silence gaps), and re-encode as a WAV file — all inside your browser. No files are uploaded to any server.

Frequently Asked Questions

Are my audio files uploaded to a server?
No. All merging is done using FFmpeg.wasm which runs entirely in your browser. Your files never leave your device and are never transmitted to any server.
What formats are supported?
The audio merger supports MP3, WAV, OGG, FLAC, and other common audio formats. You can merge files of the same format for best results.
Can I add silence between tracks?
Yes. You can insert a gap of silence between merged audio tracks, which is useful for creating podcasts, albums, or audio compilations with natural pauses between segments.
